# AirPower AI 模块

## 概述
AirPower AI模块是AirPower4J框架中的AI功能扩展模块，提供人工智能相关的功能支持。

## 功能特性
- 集成Spring Web MVC框架
- 提供AI相关基础组件
- 统一异常处理机制
- 基础编码工具支持

## 依赖关系
- `airpower-core`: 核心功能模块
- `airpower-exception`: 异常处理模块
- `spring-webmvc`: Spring Web MVC框架
- `commons-codec`: 编码解码工具
- `reflections`: 反射工具库

## 使用场景
适用于需要集成AI功能的后端应用开发，提供标准化的AI接口和服务实现。

## 配置说明
模块遵循AirPower框架的统一配置规范，支持自动装配和配置继承。
