# AirPower API 模块

## 概述
AirPower API模块是AirPower4J框架中的API开发核心模块，提供RESTful API开发的基础功能。

## 功能特性
- 集成Spring Boot Web框架
- 提供API开发基础组件
- 统一异常处理机制
- 基础编码工具支持

## 依赖关系
- `airpower-core`: 核心功能模块
- `airpower-exception`: 异常处理模块
- `spring-boot-starter-web`: Spring Boot Web启动器
- `commons-codec`: 编码解码工具

## 使用场景
适用于构建基于Spring Boot的RESTful API服务，提供标准化的API开发模式。

## 配置说明
模块遵循AirPower框架的统一配置规范，支持自动装配和配置继承。
