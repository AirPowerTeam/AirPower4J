# AirPower Exception 模块

## 概述
AirPower Exception模块是AirPower4J框架中的异常处理模块，提供统一的异常处理机制。

## 功能特性
- 统一异常处理框架
- 自定义异常类型
- 异常信息标准化
- 集成Lombok简化代码

## 依赖关系
- `airpower-core`: 核心功能模块
- `lombok`: 代码简化工具
- `jetbrains.annotations`: 注解支持
- `spring-boot-autoconfigure`: Spring Boot自动配置

## 使用场景
适用于所有需要统一异常处理的AirPower模块，提供一致的异常处理体验。

## 配置说明
模块遵循AirPower框架的统一配置规范，支持自动装配和配置继承。
